# 轻图 (QingTu) - 免费在线图片处理工具

🔗 **在线使用**: [https://tyr1105.github.io/qingtu/](https://tyr1105.github.io/qingtu/)

## 功能

### 🗜️ 图片压缩
- 自定义压缩质量（10%-100%）
- 可设置最大宽度限制
- 支持自定义输出格式
- 批量处理，最多50张

### 🔄 格式转换
- PNG ↔ JPG ↔ WebP 互转
- 批量格式转换

### 📐 调整尺寸
- 自定义宽高
- 内置常用尺寸预设（微信头像、公众号封面、小红书封面、淘宝主图等）
- 保持/取消宽高比

## 特点

- 🔒 隐私安全 - 所有处理在浏览器本地完成
- ⚡ 极速处理 - Canvas API原生处理
- 💰 完全免费 - 不限次数无水印
- 📱 多端适配 - 响应式设计

## 技术栈
React 19 + Vite + Canvas API + GitHub Pages
